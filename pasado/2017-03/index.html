PHP Day Perú

Estamos encantados de presentar la segunda edición PHP day Perú y el primero del 2017. Un día de conversatorios sorprendentes, oportunidades de contactos abundantes y mucho aprendizaje.

Queremos reunir a la comunidad PHPera del Perú para compartir nuestras experiencias utilizando PHP con diferentes frameworks, CMS, manejadores de versiones, sistemas operativos, servidores web y todo concerniente el mundo web.

  • Un día de conferencias
  • Audiencia

Tendremos un día completo de conferencias donde veremos diversos temas relacionados a la web y PHP como:

  • PHP7
  • Composer
  • Github
  • Behat
  • Mucho más ... ☺

Este evento está orientado a desarrolladores PHP y *no* PHP, emprendedores y todos quienes tengan muchas ganas de aprender, y pertenecer a una de las comunidades más grandes del Perú.
Recuerda que debes estar inscrito en la página EventBrite del evento, el día del evento debes llevar tu entrada digital o impresa y tambien tu documento de identidad.
